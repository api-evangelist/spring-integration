Spring AI 2.0.0-RC1 is now available from Maven Central as an API stabilization milestone. The release features a significant tool-calling overhaul that unifies tool execution across all supported models including OpenAI, Ollama, and Anthropic via ChatClient with ToolCallingAdvisor. Additional updates include chat memory improvements with turn-boundary snapping in MessageWindowChatMemory, structured output enhancements, DeepSeek V4 support, and an MCP SDK upgrade to 2.0.0-RC1.
